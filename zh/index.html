Awesome Starter

一个与项目模板相关的精选列表。

快速上手

简单体验 #

你可以通过包管理器的命令来简单体验，通过创建配置的方式直接创建你需要的模板。

npm create preset

然后按照提示进行操作！

如果下载失败，可通过 npm create preset proxy on 命令开启代理然后重试。

查阅 起步指南 了解更多用法。
